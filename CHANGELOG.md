# Changelog

## 1.1.1 - 2026-07-23

- 提供 7 个 BLACK SOULS MCP 工具。
- 加入启动代次、进程、帧号、命令编号和损坏快照恢复。
- 支持地图、事件、消息、菜单、队伍和战斗状态。
- 支持白名单键盘动作与连续动作。
- 在不抢占前台的情况下恢复 RPG Maker 后台键盘循环。
- 为安装、卸载和回滚配置备份使用毫秒时间与随机后缀，避免连续操作覆盖备份。
- 将源码整理为不包含游戏、存档和本机运行数据的公开仓库结构。
- 加入中英文入口文档、GitHub CI、问题模板和完整仓库元数据。
- 统一 MCP 工具的结构化错误返回。
- 集成测试会把显式运行目录传给子进程，可真实核对外部游戏副本与存档。
- 限制命令队列、序列步骤和总帧数，并严格校验 RGSS3 命令文件。
- 修正连续动作中等待帧多停一帧的问题。
- 仅在桥接心跳确实停滞时发送后台恢复通知，正常连接不再重复唤醒窗口。
- 启动桥接失败时终止本次新建的游戏进程，避免残留后台进程。
